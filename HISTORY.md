### Обязательный интервал

`scheduled_from` и `scheduled_to` теперь **обязательны**. Без них — `422`. Если `scheduled_from > scheduled_to` — тоже ошибка.

### Логика выборки

1. **One-time** — `scheduled_at` попадает в `[from, to]`, в ответе одно событие с `repetition_event_number: 0`.

2. **Повторяющиеся** (`daily`, `monthly`, `odd_even`) — шаблон берётся, если `scheduled_at <= to` (может начинаться до `from`). Для каждого шаблона генерируются вхождения в интервале.

3. **`customized_event`** — пока не участвует.

### Нумерация серии

- `0` — исходное событие (дата `scheduled_at` задачи)
- `1`, `2`, … — следующие по правилам типа

Пример: daily с `period: 2`, старт 20 мая → в интервале 24–28 мая события `#2`, `#3`, `#4` (24, 26, 28).

### Реализация

- `Tasks::Occurrence` — value object `(task, event_number, scheduled_at)`
- `generate_occurrences(from, to)` на подклассах STI
- `Tasks::Filter` собирает one-time + развёрнутые recurring, сортирует по `scheduled_at`
- `TaskSerializer` умеет сериализовать `Occurrence` (подставляет `scheduled_at` и `repetition_event_number` вхождения)

### Пример запроса

```
GET /tasks?scheduled_from=2026-05-24&scheduled_to=2026-05-28
```

### Поведение API

- **`DELETE /tasks/:id`** — `204 No Content`, без тела
- **Повторный DELETE** — `404` (one-time не находится через `default_scope`, повторение серии — через `occurrence_slot`)
- **`PATCH` удалённого повторения** — `404`, как будто слота нет
- **`GET` удалённого повторения** — по-прежнему `404` (scoped lookup)

### Код

- **`for_series_and_event`** — без `unscoped`, находит только активные кастомизации
- **`occurrence_slot`** — с `unscoped`, для проверки «tombstone» при delete/filter
- **`CustomizeOccurrence`** — удалённый слот → `404`, при `delete: true` возвращает `success(nil)`
- **`Delete`** — успех без value; повторное удаление → `404`
- Из **`Task`** убрана валидация `must_not_be_deleted` — deleted-записи просто не попадают в scope

`Filter` по-прежнему использует `unscoped` для пропуска удалённых слотов при генерации серии — это внутренняя логика, снаружи их не видно.
